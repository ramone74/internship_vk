# internship_vk

Задание 2: Путь к стиму определяется с помощью реестра Windows, поэтому будет корректно работать при любом пути установки. Для определения состояния загружаемых игр используются файлы appmanifest, оттуда извлекается StateFlag. Скорость определяется не напрямую, а через общий входящий трафик.
